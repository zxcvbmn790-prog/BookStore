Features

전체 기능 목록

dev3.3 기준 통합 기능

dev3.0
Email OTP 인증
Gmail SMTP로 6자리 OTP 발송. 3분 유효시간 내 인증 완료 시 가입 가능.
dev3.0
회원 등급 시스템
BRONZE(1%) → SILVER(2%) → GOLD(3%) → PLATINUM(5%). 누적 마일리지 기준 자동 승급.
dev3.0
마일리지 적립 & 사용
등급별 적립률 기반 마일리지 적립. 주문 시 보유 마일리지로 결제 차감.
dev3.0
배송지 자동 저장
첫 주문 후 받는사람/연락처/주소 DB 저장. 다음 주문 시 자동 입력.
dev3.0
Admin / Tracking 권한 분리
ROLE_ADMIN: 도서 CRUD, 매출, 회원관리. ROLE_TRAKING: 배송 전담.
dev3.0
카카오 ISBN 자동조회
ISBN 입력으로 카카오 API에서 도서명/저자/출판사/표지/가격 자동 입력.
dev3.0
카카오 소셜 로그인
카카오 OAuth2 연동 간편 로그인. 자동 회원가입 및 세션 연동.
dev3.0
비회원 장바구니
쿠키 기반 비회원 장바구니. 7일 유지, 로그인 없이 주문 가능.
dev3.0
실시간 1:1 채팅
WebSocket 기반 사용자-관리자 1:1 상담. 채팅 이력 DB 저장.
dev3.1
도서 할인율 시스템
관리자가 도서별 할인율(0~99%) 설정. 정가/할인가 자동 계산 및 뱃지 표시.
dev3.1
광고 도서 사이드바
관리자가 광고 도서 지정. 도서 목록 우측 사이드바에 광고 도서 노출.
dev3.1
관리자 매출 통계
일/주/월/연도별 매출 차트, 인기 도서 TOP5, 오늘 매출 실시간 집계.
dev3.2
별점 유효성 검사
별점 미선택 상태로 저장 버튼 클릭 시 클라이언트에서 차단. 400 에러 방지.
dev3.2
회원 탈퇴 비밀번호 인증
탈퇴 전 비밀번호 재확인. 카카오 간편가입 계정은 비밀번호 검증 예외 처리.
dev3.3
reCAPTCHA 자동 가입 방지
Google reCAPTCHA v2 연동. OTP 발송 전 캡차 인증 필수. 서버에서 토큰 이중 검증.
dev3.3
아이디 중복 확인 강제화
회원가입 시 중복 확인 미완료 상태에서 가입 버튼 비활성화. 중복 ID 가입 원천 차단.
